# 🧪 Testes de Software — Shopee

Este projeto tem como objetivo praticar testes manuais em uma aplicação real (Shopee), simulando atividades de um profissional de QA (Quality Assurance).

---

## 🎯 Objetivo

Validar funcionalidades, identificar falhas e garantir a qualidade do sistema através de testes manuais.

---

## 🧪 Cenários de Teste

- ✔️ Login com dados inválidos  
- ✔️ Login com campos vazios  
- ✔️ Login com e-mail inválido  

---

## 🛠️ Ferramentas utilizadas

- Navegador Web (Google Chrome)  
- GitHub para documentação  

---

## 📊 Resultados

Todos os testes realizados tiveram comportamento esperado, com validações corretas por parte do sistema.

---

## 📌 Conclusão

A aplicação apresentou boas práticas de validação de dados, impedindo acessos inválidos e melhorando a experiência do usuário.
