# Casos de Teste

## Teste 1 — Login com dados inválidos

**Cenário:** Usuário tenta fazer login com senha errada  

**Passos:**
1. Acessar a página de login  
2. Inserir e-mail válido  
3. Inserir senha incorreta  
4. Clicar em entrar  

**Resultado esperado:**
Exibir mensagem de erro informando dados inválidos  

**Resultado obtido:**
O sistema exibiu mensagem de erro informando que o e-mail ou senha estão incorretos, impedindo o login.
